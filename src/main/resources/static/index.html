The purchase application provides two REST endpoints:

  • POST /purchase/ - store a new purchase transaction provided in the request body, return a unique ID that can be later used to retrieve the purchase transaction.
  • GET /purchase/{id}?country_currency_desc={countryCurrencyDesc} - retrieve a previously-stored purchase transaction by ID, and additionally return the purchase amount converted into another currency.

Try these endpoints out using the Swagger-UI page.
